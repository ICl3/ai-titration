AI 滴定控制系统

基于计算机视觉的自动化滴定平台。将摄像头对准锥形瓶，AI 自动识别溶液颜色变化，精准控制蠕动泵完成滴定实验。

本项目由南京大学化学化工学院 三氯化碘AI滴定项目组独立开发，旨在将计算机视觉技术引入本科化学实验教学。传统手工滴定依赖人眼判断颜色变化，存在主观性强、重复性差等问题。本系统利用 深度学习 (ResNet34) 与 HSV 色彩空间分析 双模式，配合蠕动泵精准控制，覆盖酸碱滴定、配位滴定、沉淀滴定三种核心实验类型。支持多帧连续确认终点检测、DL+HSV 混合交叉验证、智能摄像头自动选择等先进功能，有效避免误判，提升滴定精度。提供本地版（Python Flask + PyTorch 深度学习）和在线版（纯浏览器 HSV 分析，无需安装）两种方案。

开始实验

酸碱滴定

甲基橙指示剂，盐酸滴定氢氧化钠。溶液由黄色变为橙色即为终点。

甲基橙 · 黄→橙

配位滴定

铬黑T指示剂，EDTA 滴定自来水。溶液由紫红色变为蓝色即为终点。

铬黑T · 紫红→蓝

沉淀滴定

铬酸钾指示剂，硝酸银滴定氯化钠。出现砖红色沉淀即为终点。

铬酸钾 · 沉淀法

使用步骤

1

插上蠕动泵和摄像头，安装 CH340 驱动

2

双击 run.bat 启动本地服务器，浏览器自动打开

3

进入实验控制页，点击「连接硬件」

4

选择实验类型和检测模式，点击「开始实验」

硬件要求

  • 兰格 GX 系列蠕动泵（兼容兰格 OEM 协议）+ CH340 USB 转串口模块
  • USB 摄像头（用于拍摄锥形瓶中溶液颜色）
  • pH 计（可选，用于电位滴定曲线记录）
  • 电脑需安装 CH340 串口驱动

技术说明

  • 双确认终点检测：可配置连续 N 帧达标才判定终点，有效避免光线波动或摄像头噪点导致的误判
  • DL+HSV 混合模式：深度学习 (ResNet34) 为主，HSV 色彩分析为辅，两者结果不一致时自动交叉验证
  • 深度学习模式：使用预训练 ResNet34 模型，针对三种滴定实验分别训练专用权重，识别精准
  • HSV 色彩分析模式：无需 GPU、无需模型，使用色彩空间阈值分析，快速轻量
  • 智能摄像头选择：自动扫描所有摄像头，通过纹理分析选择外接实验摄像头
  • 排气泡功能：实验前可快速排空管路气泡，确保滴定体积准确
  • 实时数据可视化：实验过程中实时显示滴定曲线，数据自动保存为 JSON 文件并可下载
  • 蠕动泵通信使用兰格 OEM 协议（9600bps, 8E1），支持兰格 GX 系列及兼容泵

在线版

  • 无需安装任何软件，使用 Chrome 浏览器打开即可
  • 纯浏览器 HSV 色彩分析，无需 GPU 或 Python 环境
  • 通过 Web Serial API 直接控制蠕动泵和 pH 计
